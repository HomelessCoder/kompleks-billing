# rentcharge
**Description:** Таблица, фиксирующая списания по абонентской плате

| Field | Type | Description |
|---|---|---|
| vg_id | int(11) | Идентификатор учетной записи (vg_id из vgroups) |
| dateofcharge | datetime | дата списания |
| amount | double | количество списанных средств в р.е. |
| ch_code1 | smallint(6) | тариф в соответствии с которым производилось списание средств |
| ch_code2 | tinyint(4) | сценарий списания в соответствии с которым производилось списание средств |
| ch_code3 | tinyint(4) | признак списания (регулярное списание платы/разовое списание платы) регулярное (0) – списание абонентской платы, которое производится в течение месяца, равными долями, разовое (1) – как приавило «досписание» средств в конце месяца в соответствии со сценарием №3 или 4. |
