# pre_orders
**Description:** Счета на предоплату (в сборках >=11 не используется)

| Field | Type | Description |
|---|---|---|
| ord_id | int(11) | Идентификатор счета на предоплату |
| customer | int(11) | Идентификатор плательщика (либо uid из accounts, либо vg_id – в зависимости от конвергентности) |
| summ | double | Сумма счета на предоплату |
| pay_date | date | Дата оплаты счета на предоплату |
| pay_who | int(11) | Идентификатор менеджера произведшего оплату счета (person_id из managers) |
| kvittence | varchar(15) | Номер квитанции |
| ord_date | date | Дата выставления счета на предоплату |
