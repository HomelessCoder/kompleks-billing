# vgroups
**Description:** Учетные записи

| Field | Type | Description |
|---|---|---|
| vg_id | int unsigned | Уникальный идентификатор учетной записи. |
| cu_id | int unsigned | идентификатор учетной записи во внешней системе (используется агентом IVOX) |
| c_limit | bigint | Количество байт или минут укаченных с начала месяца |
| login | varchar(63) | Логин к собственной статистике через веб |
| pass | varchar(128) | Пароль доступа к собственной статистике через веб |
| status | int(11) | Привилегированная ли группа: 1 - отключать автоматически, 0 - не отключать |
| real_balance | float(8,6) | не используется |
| blk_req | int(11) | Необходимость блокировки / разблокировки : 3 - администратором, 2 - пользовательский, 1 – по балансу, 0 – нет запроса. |
| blocked | int(11) | Признак блокировки: 5 - лимит трафика за период, 4 - активная блокировка, 3 - администратором, 2 - пользователь, 1 - баланс, 0 - разблокировано. |
| descr | text | Описание группы |
| balance | double | Баланс группы в расчетных единицах |
| tar_id | int unsigned | Номер базового тарифа (tar_id из tarifs) |
| id | int | Агент, которому принадлежит учетная запись (id из settings) |
| modified | int(11) | Признак модификации NLAI: 5 - удалена, 4 - изменена, 3 - новая, 0 - не требуется |
| indreq | int(11) | Необходимость индексации: 5 - удалена, 4 - изменена, 3 - новая, 0 - не требуется |
| b_limit | double | Предел после которого уведомлять |
| b_check | int(11) | Уведомлять ли о балансе ( если уже то 1) |
| b_notify | int(11) | Уведомлять ли о балансе |
| off_time | datetime | Время отключения группы |
| bill_active | float(8,6) | Размер оплаты за подключения к услуге |
| acc_on | bit | Флаг подключения группы к услуге |
| acc_ondate | datetime | Дата подключения услуги |
| acc_offdate | datetime | Дата предполагаемого отключения услуги |
| traff_type | int(11) | Тип трафика, подлежащего учету (1-вход, 2-исходящий, 3-общий) |
| billing_type | int(11) | Тип списания средств (0-по времени, 1-по объему трафика (для DIALUP через RADIUS) |
| depth | double | Размер кредита для абонентов RADIUS, EtherNet, NetFlow (0 - кредит не разрешен) |
| allownoip | int(11) | Разрешение работы в VPN без IP (0-запрещено, 1-разрешено) |
| tel_pin | varchar(32) | пароль при авторизации по АОН |
| changed | int(11) | Любое изменение информации о учетной записи.  5 - удалена, 4 - изменена, 3 - новая, 0 - не требуется. |
| shape | int(11) | Ограничение трафика ( 0 - не ограничено ) |
| use_aon | int(1) | Пускать по АОН в USER_NAME, проверяя пароль (tel_pin) |
| prev_c_limit | bigint(40) | Предыдущий c_limit (при смене месяца агент копирует сюда c_limit, который обнуляет) |
| zc_table | int(11) | Дата пересечения нуля балансом |
| zc_record_id | int(11) | Номер записи в таблице с детальной статистикой на которой произошло пересечение нуля |
| zc_balance | double | Баланс который был перед пересечением нуля |
| zc_type | int(11) | признак списания, вызвавшего переход баланса через 0, 1-списание агентом за употребление услуги, 2-списание абонентской платы, 3-зачисление отрицательной суммы (снятие платежа интерфейсом) |
| acc_onplandate | date | Дата подключения учетной записи |
| comments | varchar(250) | Комментарии при подключении |
| wrong_active | tinyint(3) | Счетчик неверно введенных карт предоплаты (максимально 10) |
| use_as | tinyint(1) | Флаг, использовать или нет тарификацию по AS |
| max_sessions | Int(11) | Количество одновременных сессий для данной учетной записи (для агентов RADIUS, VoIP, Кабельных) |
| wrong_date | datetime | Дата и время последнего неверного ввода карты предоплаты |
| oper_code | varchar(64) | код, определяемый оператором (для выгрузки РТ) |
| vat_free | tinyint | облагается ли НДС (для выгрузки РТ) |
| ignore_user_block | tinyint | Для учетных записей, обслуживаемых агентами RADIUS: не учитывать пользовательскую блокировку при аутентификации |
| archive | tinyint | признак удаления записи |
| d_limit | bigint | Счетчик байт (секунд для DialUp по времени), используемый для ограничения трафика за период |
| d_clear | date | Дата последнего обнуления счетчика d_limit |
