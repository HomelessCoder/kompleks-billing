# tarifs_rasp
**Description:** Расписание смены тарифных планов

| Field | Type | Description |
|---|---|---|
| record_id | int(11) | Идентификатор записи |
| vg_id | int unsigned | Учетная запись (vg_id из vgroups) нулевое значение, если учетные записи определяются по id или group_id |
| time | datetime | Дата смены тарифного плана |
| tariff_id | int(11) | Тариф, на который меняем (tar_id из tarifs) |
| id | int(11) | агент для предопределенных групп (id из settings) нулевое значение, если учетные записи определяются по vg_id или group_id |
| group_id | int(11) | объединение, для непредопределенных групп (group_id из groups) нулевое значение - уч. записи по vg_id или id |
| changed | tinyint | Изменен ли тариф (0-нет, 1-да). |
| request_by | tinyint | запрос на смену тарифа (0 – администратор/менеджер, 1 – клиент из клиентского интф) |
| old_tariff_id | Int(11) | Текущий тариф на момент выполнения смены. |
