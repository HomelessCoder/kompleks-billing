# templates
**Description:** Шаблоны документов

| Field | Type | Description |
|---|---|---|
| template_id | int(11) | Порядковый номер шаблона |
| document_type | int(3) | Тип документа для которого создан шаблон (1-Договор, 2-Приложение, 3-Доп Соглашение, 4-Для пользователей) |
| account_type | int(1) | Тип абонента (0-Любой, 1-Физ лицо, 2-Юр Лицо) |
| agent_id | int(11) | Идентификатор агента для уч записей которого будет доступен данный документ |
| template_filesize | int(11) | Размер файла шаблона |
| template_name | varchar(30) | Название шаблона |
| template_filename | varchar(20) | Имя файла шаблона |
| original_filename | varchar(100) | Изначальное (оригинально имя файла шаблона) |
