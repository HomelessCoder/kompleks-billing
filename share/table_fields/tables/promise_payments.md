# promise_payments
**Description:** Обещанные платежи

| Field | Type | Description |
|---|---|---|
| record_id | int | Порядковый номер записи |
| uid | int | Идентификатор пользователя, заказавшего платеж |
| vg_id | int | идентификатор учетной записи, заказавшей платеж (не конвергентный режим) |
| amount | double | заявленная величина платежа |
| prom_date | datetime | время «обещания» платежа |
| prom_till | datetime | погасить платеж до указанной даты |
| debt | double | непогашенный остаток обещанного платежа |
| pay_id | int | идентификатор платежа (в bills), которым был погашен обещанный платеж. Если платежей было несколько, то отражается последний, если платеж просрочен, то -1 |
