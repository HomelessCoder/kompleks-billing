# __ccperiodiс
**Description:** "Внешняя" таблица с данными о переодических услугах (агент IVOX)

| Field | Type | Description |
|---|---|---|
| record_id | int unsigned | идентификатор записи |
| cu_id | int unsigned | идентификатор учетной записи во внешней системе |
| timefrom | datetime | дата начала действия услуги |
| cat_id | int(11) | номер каталога переодических услуг |
| service_id | int(11) | идентификатор услуги из каталога |
| scenary_id | int(11) | сценарий списания абонентской платы (1,2, и т.д.) |
| mul | double | оказать услугу в количестве... (множитель для стоимости из каталога) |
