# h_tiket_status
**Description:** Статусы хелпдеска

| Field | Type | Description |
|---|---|---|
| status_id | int(11) | Номер статуса |
| status_type | tinyint(4) | Тип статуса (0-для всех, 1-для менеджера, 2-для пользователя) |
| status_name | varchar(100) | Название статуса |
| status_color | varchar(7) | Цвет, Которым отображается статус |
| status_act | tinyint(1) | Признак активности статуса (возможность присвоения данного статуса тикету) |
| status_default | tinyint(1) | Признак того, что тикет будет помечен как отображаемый в фильтре |
| status_answer | tinyint(1) | Признак доступности ответа на этот тикет |
| status_new | tinyint(1) | Признак того, что данный статус будет присвоен только что созданному тикету |
| st_for_answer | tinyint(1) | Признак того, что данный статус будет присвоен тикету после ответа на него |
