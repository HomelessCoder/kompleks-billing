# route_weights
**Description:** Весовые коэффициенты динамических маршрутов для VoIP

| Field | Type | Description |
|---|---|---|
| cat_num | int(11) | номер каталога |
| cat_idx | int(11) | номер категории каталога |
| route_id | int(11) | идентификатор маршрута (совпадает с record_id в oper_staff) |
| weight | int(11) | вес (приоритет) маршрута |
