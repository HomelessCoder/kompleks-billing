# cat_N_idx
**Description:** Тарифные категории каталога (эта таблица не используется для каталога разовых услуг) N – cat_num из catalog

| Field | Type | Description |
|---|---|---|
| cat_idx | int(11) | категория |
| cat_zonecode | varchar(30) | мнемонический код тарифной категории |
| cat_descr | varchar(255) | описание тарифной категории |
| cat_above | int(30) | стоимость звонка (р.е./мин для телефонии, р.е./Мб для интернета) |
| zone_mode | int(11) | Переключатель, 1-оплата по времени; 2-оплата по факту звонка. |
| cat_rent | tinyint(1) | Флаг, входит ли категория в направление, на которое расходуются средства абонентской платы |
| deleted | tinyint(1) | Признак удаленной категории (категории физически не удаляются) |
| oper_above | int(11) | Себестоимость |
| type | int(11) | категория звонка из calltypes |
