# orders
**Description:** Счета… и прочие отчетные документы

| Field | Type | Description |
|---|---|---|
| order_id | int(11) | Идентификатор счета |
| user_id | int | для конвергентного режима - uid, для неконвергентного - vg_id. |
| uid | int(11) | Идентификатор пользователя (uid из accounts) (не используется в сборках >=11) |
| start_date | date | Первое число отчетного месяца (для счетов на предоплату =NULL) |
| end_date | date | дата генерации счета |
| order_date | date | Дата выставления счета (дата, указанная на документе) |
| nds_summ | double | Сумма НДС счета (в нац. Валюте) |
| comm_summ | double | Сумма счета (в нац. Валюте) |
| re_summ | double | сумма счета в р.е. (используется для оплаты) |
| order_num | int(11) | номер счета (документа) |
| oper_id | int | идентификатор оператора, реквизиты которого указывать в счете |
| doc_id | int | идентификатор документа из documents |
| pay_date | date | Дата оплаты счета |
| pay_who | int(11) | Идентификатор менеджера произведшего оплату счета (person_id из managers) |
| kvitence | varchar(15) | Номер квитанции |
| fact_num | int(11) | Номер счета-фактуры |
| fact_date | date | Дата выставления счета-фактуры |
