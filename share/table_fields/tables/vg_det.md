# vg_det
**Description:** Таблица с описанием детализации трафика по каждой группе

| Field | Type | Description |
|---|---|---|
| vg_id | int(11) | Учетная запись (vg_id из vgroups) |
| vg_det | int(11) | Информация об удаленном ресурсе (0 - нет 1 - да) |
| id | int(11) | Агента (id из settings) |
| remoteinfo | int(11) | Информация о сервисе |
