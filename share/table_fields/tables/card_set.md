# card_set
**Description:** Наборы карт оплаты

| Field | Type | Description |
|---|---|---|
| set_id | Int unsigned | Идентификатор набора карт оплаты |
| set_descr | Varchar(80) | Описание набора карт |
| traff_type | tinyint | тарифицируемое направление трафика (аналогично параметру в vgroups) |
| created_by | int(11) | Менеджер, создавший набор |
