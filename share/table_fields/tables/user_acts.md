# user_acts
**Description:** Не используется в сборках >=11

| Field | Type | Description |
|---|---|---|
| act_id | int(11) | уникальный идентификатор акта |
| vg_id | int unsigned | идентификатор учетной записи для неконвергентного режима |
| uid | int(11) | идентификатор пользователя для конвергентного режима |
| act_date | date | дата акта |
| oper_id | int(11) | идентификатор оператора |
