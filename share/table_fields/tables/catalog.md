# catalog
**Description:** Описание каталогов

| Field | Type | Description |
|---|---|---|
| cat_id | int(11) | Идентификатор каталога (каталог размещается в талицах cat_[cat_id], cat_[cat_id]_idx) |
| cat_name | char(64) | Имя каталога |
| cat_type | int(11) | Тип каталога: 0 - телефонный, 1 - IP, 2 - AS, 3 - разовые услуги, 4 - периодические |
| cat_recnum | int(11) | Количество записей в каталоге |
| cat_owner | char(64) | Владелец каталога |
| reread | datetime | Время последнего изменения каталога |
| oper_id | int(11) | Идентификатор оператора, которому принадлежит каталог |
