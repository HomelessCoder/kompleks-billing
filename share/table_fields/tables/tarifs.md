# tarifs
**Description:** Тарифы

| Field | Type | Description |
|---|---|---|
| tar_id | int(11) | Идентификатор тарифа |
| includes | int(11) | Предоплаченный трафик для базового тарифа для кабельных агентов |
| above | int(11) | Стоимость превышения (за Мб для интернета, за минуту локального звонка для телефонии) |
| spec | int(11) | Признак специального тарифа (идентификатор) |
| rent | int(11) | Аренда канала (стоимость) |
| block_rent | int(11) | Аренда для уч. Записи в заблокированном состоянии (стоимость) для 2го и 3го сценариев |
| descr | text | Описание тарифа |
| rent_mode | int(1) | Сценарий списания абонентской латы (0 – 4). |
| type | int(2) | Тип тарифа (0-выделенная линия(Netflow,Ethernet,sFlow),1-Dialup(Radius),2-PABX(PCDR),3-для совместимости – тоже что и 2,4-IVOX, 5-VoIP, 6-Dialup(Radius – по времени)) |
| use_includes | int(11) | Стоимость звонков и услуг включенных в абон. плату |
| free_seconds | int(11) | Количество бесплатных секунд для ВСЕХ тарифных зон в этом тарифе. |
| conf_above | int(11) | Стоимость минуты конференции в агенте PCDR, в р.е. Снимается только с модератора конференции. |
| call_mode | int(11) | Тарификация локальных звонков по : 0 - времени, 1 - по факту. |
| act_block | int(11) | Применять активную блокировку ( 1 - Да, 2 - Нет, 0 - не применимо ) |
| round_seconds | int(11) | Точность округления |
| cat_number | int(11) | Номер каталога для тарификации |
| incoming_cost | int(11) | Стоимость входящего звонка |
| redirect_cost | int(11) | Стоимость переадресации звонка |
| ivr_charge | int(11) | Стоимость работы с IVR |
| voicemail_charge | int(11) | Стоимость работы с голосовой почтой |
| opcall_charge | int(11) | Стоимость работы с оператором |
| cat_number_ivox | int(11) | Номер каталога разовых услуг |
| cat_number_incoming | int(11) | Номер каталога для тарификации входящих звонков |
| local_round_seconds | int(11) | Точность округления локальных звонков (сек) |
| local_free_seconds | int(11) | Бесплатное время локальных звонков (сек) |
| emptycall_charge | int(11) | Цена нулевого исходящего звонка |
| shape | Int(11) | скорость для шейпера |
| archive | tinyint | 0 – активный тариф, 1 – удаленный (заархивированный тариф) |
| voip_block_local | tynyint | 1 - означает блокировать локальные звонки Это поле только для агента VoIP |
| cat_number_ivox_per | int(11) | идентификатор каталога периодических услуг для тарифов IVOX |
| price_plan | tinyint(4) | тип тарифа |
| dyn_route | tinyint(4) | Динамическая маршрутизация VoIP звонков: 0-не использовать, 1-LCR, 2-MWR |
| reverse_incoming | tinyint unsigned | При тарификации входящих звонков по каталогу поиск в каталоге осуществляется по номеру Б (1) или по номеру А (0) |
| traff_limit | int | Ограничение трафика в Мб (Мин. Для DialUP по времени) за период traff_limit_per |
| unavaliable | tinyint | 1 — скрыть в списках назначения для назначения, 0 — доступен для назначения |
| traff_limit_per | int | Период в днях, на который вводится ограничение по трафику. Значение 0 означает календарный месяц. |
