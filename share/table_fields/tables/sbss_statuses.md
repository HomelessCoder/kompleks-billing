# sbss_statuses
**Description:** Статусы запросов HelpDesk

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| type | int | Класс доступа к запросу с этим статусом: 1 — всем, 2 — менеджеру, 3 — пользователю |
| descr | varchar | Описание/Название статуса |
| color | varchar | Цвет (hex) которым подсвечивать в списках |
| active | tinyint | 1- доступен для назначения, 0 — не доступен |
| display_default | tinyint | Отображать по умолчанию в списке административного интерфеса |
| client_modify_allow | tinyint | 1 — пользователь может добавлять сообщение в запрос с этим статусом |
| default_new | tinyint | Новое сообщение от клиента сохранить с id этого статуса |
| default_answer | tinyint | Статус ответа от клиента |
