# managers
**Description:** Менеджеры и их полномочия

| Field | Type | Description |
|---|---|---|
| person_id | int(11) | идентификатор менеджера |
| login | varchar(32) | имя учетной записи |
| pass | varchar(32) | пароль доступа к административной консоли |
| fio | varchar(240) | фамилия, имя, отчество |
| email | varchar(64) | электронная почта |
| descr | text | описание менеджера |
| office | varchar(32) | номер офис |
| agents | bit | признак доступа к настройке сетевых агентов (1 - да, 0 - нет) |
| options | bit | признак доступа к опциям |
| tars | bit | признак доступа к тарифам |
| groups | bit | признак доступа к настройке учетных записей |
| managers | bit | признак доступа к управлению менеджерами |
| commstat | bit | признак доступа к общей статистике по выделенным каналам |
| detstat | bit | признак доступа к детальной статистике по выделенным каналам |
| dialstat | bit | признак доступа к статистике DialUp пользователей |
| payments | bit | признак доступа к работе с оплатой счетов |
| paylist | bit | признак доступа к списку счетов |
| factlist | bit | признак доступа к списку счетов-фактур |
| salebook | bit | признак доступа к книге продаж |
| help | bit | признак доступа к оперативной справке |
| rate | bit | признак доступа к работе с курсом национальной валюты |
| usergroups | bit | признак доступа к работе с объединениями учетных записей |
| reportlist | bit | признак доступа к меню подготовки отчетов |
| catalog | bit | признак доступа к работе с каталогом (IP групп, телефонных кодов) |
| ph_stat | bit | признак доступа к телефонной статистике |
| logs | bit | признак доступа к журналу событий |
| dialsess | bit | признак доступа к просмотру активных DialUp сессий |
| ph_ip | bit | признак доступа к VoIP статистике |
| users | bit | признак доступа к настройке пользователей |
| accountgroups | bit | признак доступа к настройке групп пользователей |
| cards | bit | Признак доступа к просмотру активированных карт предоплаты |
| gen_card | bit | Признак доступа к генерации карт предоплаты |
| gen_ords | bit | Признак доступа к генерации счетов |
| gen_reps | bit | Признак доступа к генерации отчетов |
| exp_ords | bit | Признак доступа к экспорту  счетов |
| vol_stat | bit | Признак доступа к объемной статистике |
| time_stat | bit | Признак доступа к временной статистике |
| per_stat | bit | Признак доступа к периодической статистике |
| once_stat | bit | Признак доступа к разовой статистике |
| orders | bit | Признак доступа к счетам |
| documes | bit | Признак доступа к шаблонам документов |
| allow_hd | bit | Признак доступа к HelpDesk |
| gen_tolink | bit | Признак доступа к генерации карт на подключение |
| operator | tinyint(1) | Признак доступа к статистике по операторам |
| recalc_bit | tinyint(1) | Признак доступа к выполнению отката |
| platezi | tinyint(1) | Признак доступа к просмотру платежей |
| calendar | tinyint(1) | Признак доступа к редактированию календаря |
| gen_docs | bit | Признак доступа к генерации шаблонов документов |
| rad_attr | Bit | Признак доступа к радиус-атрибутам |
| card_set | tinyint(1) | Признак доступа к наборам карт |
| users_doc_allow | tinyint(1) | Признак доступа к формированию документов (шаблоны) в свойствах пользователя |
| users_pre_orders | tinyint(1) | Признак доступа к созданию документов на предоплату в свойствах пользователя |
| services | tinyint(1) | Признак доступа к меню Услуги и возможность просмотра / назначения услуг в учетной записи IVOX |
| broadcast | tinyint(1) | Признак доступа к широковещательным сообщениям |
