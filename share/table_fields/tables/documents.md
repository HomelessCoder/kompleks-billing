# documents
**Description:** Пользовательские отчетные документы

| Field | Type | Description |
|---|---|---|
| doc_id | int | идентификатор документа |
| name | varchar(64) | Название документа |
| doc_schema | tinyint | схема выставления: 0 - кредитная, 1 - авансовая, 2 - смешанная |
| nds_above | tinyint | Документ подразумевает НДС "в том числе" (0) или сверху (1) |
| template | varchar(255) | имя файла с шаблоном |
| save_path | varchar(255) | Путь к директории для сохранения генерируемых документов |
| usergroup_id | int | При автоматической генерации в конце месяца идентификатор группы пользователей, для которой необходимо сгенерировать документ. |
| group_id | int | При автоматической генерации в конце месяца идентификатор объединения, для которого необходимо сгенерировать документ. Если usergroup_id=group_id=0 - генерировать для всех, если usergroup_id=group_id=NULL - не выполнять автоматической генерации. |
| payable | tinyint | Документ подразумевает оплату: 0 - нет, >0 - да. 1 - счета с постоплатой,  2 - счета на предоплату. |
| upload_ext | varchar(64) | mime type для документа (используется при отправки в браузер) |
| client_allowed | tinyint | Доступен ли в клиентском интерфейсе (0 - не доступен). |
| hidden | tinyint | 1 — скрывать шаблон документа в списках при генерации и отображении, 0 — показывать (эквивалент удаления шаблона). |
| onfly | tinyint | 0 – счет, 1 – отчет (отчет не сохраняется в orders и не генерируется автоматически) |
| detail | tinyint | 0 – за месяц, исаользуя reportYYYYMM, 1 – любой интервал времени |
