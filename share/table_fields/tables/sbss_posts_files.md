# sbss_posts_files
**Description:** Данные о прикрепленных файлах к сообщениям запросов (HelpDesk)

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| ticket_id | int | идентификатор запроса, привязан внешним ключом к sbss_tickets.id |
| post_id | int | идентификатор сообщения к которому привязан файл, привязан внешним ключом к sbss_posts.id |
| author_type | tinyint | тип автора создавшего запрос (0 — менеджкер, 1 — клиент) |
| author_id | id | идентификатор автора в зависимости от типа |
| created_on | datetime | дата создания сообщения |
| edited_on | datetime | дата редактирования |
| edited_type | tinyint | тип автора изменившего запрос (0 — менеджкер, 1 — клиент) |
| edited_id | int | идентификатор автора в зависимости от типа |
| size | int | размер прикрепленного файла в байтах |
| description | varchar | описание прикрепленного документа |
