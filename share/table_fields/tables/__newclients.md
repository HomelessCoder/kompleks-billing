# __newclients
**Description:** Таблица импорта новых пользователей (IVOX)

| Field | Type | Description |
|---|---|---|
| cu_id | int unsigned | Уникальный идентификатор клиента во внешней системе |
| status | int(11) | Привелегированная ли группа: 1 - отключать автоматически, 0 - не отключать |
| blocked | int(11) | Признак блокировки: 3 - администратором, 2 - пользователь, 1 - баланс, 0 - разблокировано |
| act_status | int(11) | Состояние пользователя (0 – действующий или 1 - отключенный) |
| balance | double | Баланс пользователя, если не нулевой |
| forname | varchar(32) | Имя контактного лица |
| interimname | varchar(32) | Отчество контактного лица |
| familyname | varchar(32) | Фамилия контактного лица |
| type | tinyint | Тип пользователя: 1 - Юридическое лицо, 2 - Физическое лицо |
| officialname | varchar(32) | Название компании (заполняется если клиент Юр. Лицо) |
| tel_pin | varchar(32) | PIN |
| login | varchar(20) | Логин к собственной статистике через веб (в CRM это цифровой идентификатор клиента) |
| pass | varchar(20) | Дублирование поля PIN |
