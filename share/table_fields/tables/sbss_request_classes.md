# sbss_request_classes
**Description:** Классы запросов в HelpDesk

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| descr | varchar | Описание/Название класса запроса |
| color | varchar | Цвет (hex) которым подсвечивать в списках |
| responsible_man | int | идентификатор менеджера назначаемого по умолчанию для нового запрос этого класса |
