# calltypes
**Description:** категории звонков классифицируемых модулем Lbsupervise

| Field | Type | Description |
|---|---|---|
| id | int(11) | идентификатор типа |
| name | varchar(32) | название типа |
| descr | varchar(255) | описание типа |
| flags | int(10) | 1 – включает дополнительное поле в статистике (трафик), 0 – столбцы по умолчанию |
| color | int(10) | RGB цвет для отображения в статистике и каталогах |
| extern_code | varchar(32) | код услуги для выгрузки совинтел |
