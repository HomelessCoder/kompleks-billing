# we_bonus
**Description:** Определяет скидку в выходной день

| Field | Type | Description |
|---|---|---|
| tar_id | int(11) | Тариф (tar_id из tarifs) |
| rate | int(11) | стоимость превышения (в абсолютном значении) |
| discount | double | относительная скидка (процент от ставки базового тарифа) |
| shape_discount | double | Относительное уменьшение пропускной способности (относительно значения в общем тарифе) |
| shape_rate | int | Абсолютное значение пропускной способности |
