# bonuses
**Description:** Скидки по времени

| Field | Type | Description |
|---|---|---|
| tar_id | int unsgned | Тариф (tar_id из tarifs) |
| timefrom | time | Время начала действия скидки |
| timeto | time | Время окончания действия скидки |
| discount | double | Относительный размер скидки (-1 – абсолютная скидка) |
| rate | int | Абсолютный размер скидки (-1 – скидка относительная) |
| shape_discount | double | Относительное уменьшение пропускной способности (относительно значения в общем тарифе) |
| shape_rate | int | Абсолютное значение пропускной способности |
| bonus_id | int unsigned | Идентификатор скидки |
