# h_post
**Description:** Таблица сообщений внутреннего хелпдеска

| Field | Type | Description |
|---|---|---|
| post_id | int(12) | Порядковый номер сообщения |
| post_date | datetime | Дата добавления сообщения |
| id | int(12) | Идентификатор тикета, к которому относится сообщение |
| number | int(11) | Порядковый номер сообщения в данном тикете |
| edited | tinyint(1) | Признак редактирования сообщения |
| last_edited | datetime | Дата последнего редактирования |
| post_ip | int(12) | IP-адрес с которого было добавлено сообщение |
| post_who | tinyint(1) | Флаг, показывающий кто добавил сообщение (1-менеджер, 0 - пользователь) |
| who_id | int(11) | Идентификатор либо пользователя либо менеджера (в зависимости от флага post_who) |
| post_body | text | Текст сообщения |
| post_files | varchar(50) | Имя прикрепленного файла (пока не реализовано) |
| type | tinyint(1) | Тип сообщения (0-простое сообщение, 1-служебное, 2-изменение статуса либо ответственного менеджера) |
