# h_tiket
**Description:** Тикеты внутреннего хелпдеска

| Field | Type | Description |
|---|---|---|
| id | int(12) | Порядковый номер тикета |
| create_date | datetime | Дата создания тикета |
| uid | int(11) | Идентификатор менеджера либо пользователя создавшего тикет (зависит от флага us_o_man) |
| status_id | tinyint(4) | Идентификатор статуса присвоенного данному тикету |
| tiket_title | varchar(250) | Заголовок тикета |
| whois | int(11) | Идентификатор менеджера , которому принадлежит данный тикет (ответственный) |
| us_o_man | tinyint(1) | Признак того, кто создал тикет (0-Менеджер, 1-Пользователь) |
| type | tinyint(1) | Тип (0-обычный тикет, 1-служебный) |
| tosort | int(12) | Поле актуальности тикета (чем больше число, тем выше в списке этот тикет будет) |
