# stab
**Description:** Даты последних выгрузок в отчетные таблицы (в unixtime)

| Field | Type | Description |
|---|---|---|
| lday | bigint(14) | Дата последних ежедневных действий сервера |
| lweek | bigint(14) | Дата последних ежедневных действий сервера, проводимых с часовым отставанием |
| lmonth | bigint(14) | Дата последних ежемесячных действий сервера (списание а/п) |
| lyear | bigint(14) | Дата последних ежегодных действий сервера |
| lmonth2 | bigint(14) | Дата последних ежемесячных действий сервера (выгрузка месячной статистики) |
