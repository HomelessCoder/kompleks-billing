# pay_cards
**Description:** Карты оплаты

| Field | Type | Description |
|---|---|---|
| ser_no | bigint(20) | серийный номер карты (логин для карт активации) |
| card_key | varchar(32) | ключ карты (пароль для карт активации) |
| sum | double | номинал в у.е. |
| date_create | datetime | дата создания |
| act_til | datetime | активировать до … |
| activated | datetime | дата активации |
| expire_period | int unsigned | срок действия карты после активации в сутках |
| used | tinyint | признак использованной карты (0-не исп., 1-активирована агентом, 2-израсходована для пополнения счета) |
| vg_id | int unsigned | идентификатор уч. Записи, созданной при активации карты (для карт оплаты - кем активирована) |
| mod_pers | int(11) | Менеджер, сгенеривший карту |
| card_set | int unsigned | Идентификатор набора, которому принадлежит карта |
