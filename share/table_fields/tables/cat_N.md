# cat_N
**Description:** Список сетей / кодов городов / разовых услуг N – cat_num из catalog. Набор полей может варьироваться в зависимости от типа каталога.

| Field | Type | Description |
|---|---|---|
| zone_id | int(20) | идентификатор записи в таблице |
| zone_num | varchar(30) | IP адрес / код города / код разовой услуги |
| zone_mask | varchar(16) | Маска сети (поле только в каталоге сетей) |
| tar_zone | int(20) | категория (cat_idx из cat_N_idx) |
| region | varchar(255) | регион (поле только в каталоге кодов городов) |
| region_descr | varchar(255) | описание (поле только в каталогах сетей и кодов городов) |
| descr | varchar(255) | описание (поле только в каталоге разовых услуг) |
| zone_price | int(11) | цена разовой услуги в РЕ (поле только в каталоге разовых услуг) |
| zone_rent | tinyint(4) | включать ли оплату разовой услуги в а/п (1- включать) (поле только в каталоге разовых услуг) |
