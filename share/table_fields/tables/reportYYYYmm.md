# reportYYYYmm
**Description:** Аггрегированные данные со списаниями за отчетный период

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| uid | int | уникальный идентификатор пользователя |
| vg_id | int | Уникальный идентификатор учетной записи. |
| oper_id | int | идентификатор оператора, которым оказана услуга |
| tar_id | int | идентификатор тарифа |
| cat_num | int | идентификатор каталога |
| cat_idx | int | идентификатор категории каталога |
| rent | tinyint | признак включения в а/п |
| size_discount | int | идентификатор скидки по объему |
| time_discount | int | идентификатор скидки по времени |
| we_discount | tinyint | признак наличия скидке выходного дня |
| type | tinyint | тип услуги: 1 - объемная, 2 - временная, 3 - разовая, 4 – периодическая, NULL – для абон. Платы |
| volume | double | величина услуги (байты, секунды, разы - для объемных, временных и разовых+периодических соответсвенно) |
| amount | double | величина списания (либо начислений в случае записи о платеже) |
