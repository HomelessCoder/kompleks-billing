# usergroups
**Description:** группы пользователей

| Field | Type | Description |
|---|---|---|
| group_id | int(11) | идентификатор группы |
| name | varchar(32) | название группы |
| description | text | описание группы |
| fiz_kvit | tinyint | 1 - выставлять квитанции физ. лицам вместо счетов (не используется) |
| promise_allow | tinyint | разрешить пользователям группы заказывать «обещанный платеж» через личный кабинет |
| promise_max | double | максимальная сумма «обещанного платежа» в р.е. |
| promise_min | double | минимальная сумма «обещанного платежа» в р.е. |
| promise_rent | tynyint | ограничение суммы «обещанного платежа» (сверху) величиной абон. платы |
| promise_till | int | срок для оплаты «обещанного платежа» (в днях) |
| promise_limit | double | максимальная допустимая задолженность (в р.е.) для использования услуги «обещанный платеж» |
