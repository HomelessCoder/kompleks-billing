# oper_staff
**Description:** Связка операторов с транковыми группами

| Field | Type | Description |
|---|---|---|
| id | int(11) | id агента |
| oper_id | int(11) | Идентификатор оператора |
| trunk | varchar(32) | признак оператора (транк/линия, связанная с данным оператором) |
| token_type | tinyint | тип признака оператора (0 - признак передается в спец. Поле CDR, 1 - признак содержится в номере, 2-маршрут для VoIP) |
| record_id | int(11) | Уникальный идентификатор записи. |
