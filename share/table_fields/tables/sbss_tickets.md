# sbss_tickets
**Description:** Таблица запросов helpdesk-a

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| name | varchar | тема запроса |
| author_type | tinyint | тип автора создавшего запрос (0 — менеджкер, 1 — клиент) |
| author_id | int | идентификатор автора в зависимости от типа |
| created_on | datetime | дата создания запроса |
| respondent_type | tinyint | автор последнего сообщения в по теме (0 — менеджер, 1 — клиент) |
| respondent_id | int | идентификатор последнего ответившего в по теме запроса в зависимости от типа |
| lastpost | datetime | дата последнего сообщения по теме запроса |
| class_id | int | привязка к классу запроса |
| replies | int | количество ответов со стороны административного интерфейса |
| status_id | int | Текущий статус запроса |
| manager_lock | int | идентификатор менеджера заблокировавшего тикет для написания сообщения |
| vgid | int | привязка к учетной записи |
