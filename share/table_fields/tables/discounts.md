# discounts
**Description:** Скидки по объему (льготы)

| Field | Type | Description |
|---|---|---|
| tar_id | int(11) | Тариф (tar_id из tarifs) |
| amount | int(11) | Количество сервиса после которого начинается скидка |
| discount | double | Относительный размер скидки (-1 – абсолютная скидка) |
| rate | int(11) | Абсолютный размер скидки (-1 – скидка относительная) |
| shape_discount | double | Относительное уменьшение пропускной способности (относительно значения в общем тарифе) |
| shape_rate | int | Абсолютное значение пропускной способности |
| dis_id | int(11) | Идентификатор скидки |
| bonus | int(11) | Величина зачисления на счет по достижению объема указанного в льготе |
