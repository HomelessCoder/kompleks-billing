# sbss_posts
**Description:** Таблица сообщений внутреннего хелпдеска

| Field | Type | Description |
|---|---|---|
| id | int | уникальный номер записи |
| ticket_id | int | идентификатор запроса, привязан внешним ключом к sbss_tickets.id |
| author_type | tinyint | тип автора создавшего запрос (0 — менеджкер, 1 — клиент) |
| author_id | id | идентификатор автора в зависимости от типа |
| created_on | datetime | дата создания сообщения |
| text | text | текст сообщения |
| spec | tinyint | тип сообщения 0 — общее, 1 — служебное |
