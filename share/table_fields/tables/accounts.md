## Общие поля
| Field | Type | Description |
|---|---|---|
| uid | int | Идентификатор пользователя |
| login | varchar(32) | Логин для входа в клиентский интерфейс |
| pass | varchar(32) | Пароль для входа в клиентский интерфейс |

| type | int(3) | Тип пользователя: 1 - Юридическое лицо, 2 - Физическое лицо |
| balance | double | Общий баланс (PRO, конвергентный биллинг) |
| depth | double | разрешенный размер кредита в р.е. (только для конвергентного режима). |
| descr | text | Описание |
| name | varchar(64) | Имя пользователя / Название компании |
| phone | varchar(32) | Телефон |
| fax | varchar(32) | Факс |
| agrm_num | varchar(64) | Номер договора |
| agrm_date | varchar(32) | дата договора |
| email | varchar(128) | E-mail |
| kod_1c | Char(8) | Код 1С |
| bill_delivery | tinyint | Способ доставки счета: 0-курьер, 1-почтой,2-самостоятельно, 3-другой |
| сategory | tinyint | 0-Бюджет, 1-хозрасчет,2-население, 3-ПБОЮЛ |
| last_mod_date | timestamp | дата последнего изменения (заполняется автоматически) |
| wrong_active | tinyint | Счетчик неверно введенных карт предоплаты (максимально 10) |
| wrong_date | datetime | Дата и время последнего неверного ввода карты предоплаты |
| diplomat | tinyint | 1- дипломат, 2 – не дипломат |
| resident | tinyint | 0 – не резидент, 1 – резидент |
| agrm_type | tinyint | тип договора: 1-клиент(бизнес),2-дистрибьютор(реселлер),3-партнер(комиссионер),4-оператор(взаимодействие),5-оператор(присоединение),6-дистрибьютор(агент),7-поставщик(оператор),8-клиент(частный),7-поставщик(не оператор) |
| oksm | int(11) | код страны абонента по ОКСМ |
| okato | varchar(32) | место заключения договора по ОКАТО |
| public_agree | tinyint unsigned | факт согласия абонента на использовании сведений в инф.-справочных системах |
